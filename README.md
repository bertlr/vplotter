# Install #

- install raspian
- install git:

    sudo apt-get install git-core


- install wiringPi from http://wiringpi.com/download-and-install/

    git clone https://github.com/bertlr/vplotter.git
    cd vplotter
    make

run the plotter as root:

    sudo dist/Release/GNU-Linux/vplotter
